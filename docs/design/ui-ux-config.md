# UI/UX 설정 — 글로벌 컨벤션 (Global Conventions)

업계 표준을 따르는 기본 가이드라인입니다. 모든 UI 구현은 이 문서를 원천 데이터(SoT)로 사용합니다.

---

## 1. 컬러 팔레트 (Color Palette)

### UI 컬러 (8개 토큰)

| 토큰 | Hex | 사용처 |
|-------|-----|-------|
| `UI_BG_DEEP` | `#0D1B2A` | 패널 배경, 오버레이 딤(Dim) 레이어 |
| `UI_BG_MID` | `#1A2F45` | 카드 표면, 팝업 내부 배경 |
| `UI_BORDER` | `#2A4A6B` | 패널 테두리, 구분선 |
| `UI_PRIMARY` | `#4A90D9` | 주요 액션 버튼 (Play, 확인) |
| `UI_CTA` | `#E8A020` | 골드 소비 / 구매 버튼, 별, 코인 아이콘 |
| `UI_SUCCESS` | `#3DBE6E` | 클리어 피드백, 긍정 상태, 임계값 이상의 비율 바 |
| `UI_DANGER` | `#E84040` | 실패 상태, 경고, 임계값 미만의 비율 바, 골드 부족 |
| `UI_TEXT` | `#F0EAD6` | 기본 텍스트 |

## 2. 타이포그래피 (Typography)

### 폰트 선택

| 용도 | 폰트 | 제약 사항 |
|-----|------|------------|
| 대형 헤더 (스테이지 번호, 결과 제목) | 픽셀 폰트 — `m5x7` (무료, 가독성 좋음) | 24px 이상에서만 사용 |
| 본문, 라벨, 버튼 텍스트 | 가독성 좋은 산세리프 — `Noto Sans` | 모든 크기 |
| 강조된 숫자 (턴, 골드, 비율) | 픽셀 폰트 또는 Bold 산세리프 | 크고 굵게 표시 |

**모바일 환경에서 20px 미만의 픽셀 폰트 사용 금지 — 가독성 저하.**

### 크기 스케일 (dp)

| 역할 | 크기 | 굵기 |
|------|------|--------|
| 화면 제목 | 28–32dp | Bold |
| 섹션 헤더 | 20–24dp | Bold |
| 버튼 텍스트 | 18–20dp | Bold |
| 본문 / 라벨 | 16–18dp | Regular |
| 보조 정보 | 14dp | Regular |
| 최소 허용 크기 | **12dp** | — |

## 3. 터치 타겟 (Touch Targets)

| 규칙 | 값 |
|------|-------|
| 최소 터치 영역 | **48×48dp** (구글 머티리얼 표준) |
| 주요 액션 버튼 (Play, 확인, 재시도) | **56×56dp** 권장 |
| 인접한 탭 가능 요소 간 최소 간격 | **8dp** |

---

## 4. 애니메이션 타이밍 (Animation Timing)

### 표준 소요 시간

| 유형 | 시간 | 이징(Easing) |
|------|----------|--------|
| 씬 전환 (페이드 / 슬라이드) | 250ms | ease-in-out |
| 팝업 / 오버레이 등장 | 200ms | ease-out |
| 팝업 / 오버레이 퇴장 | 150ms | ease-in |
| 버튼 피드백 | 80ms | ease-in-out |
| 별 팝업 애니메이션 | 별당 300ms | ease-out-back (약간의 오버슛) |
| 골드 카운트업 | 600ms | ease-out |

---

## 5. 여백 시스템 (Spacing System)

8dp 그리드 시스템을 사용합니다. 모든 값은 4의 배수여야 합니다.

| 값 | 용도 |
|-------|-----|
| 4dp | 아이콘 ↔ 라벨 간격, 좁은 내부 여백 |
| 8dp | 기본 요소 간격 |
| 16dp | 패널 내부 여백, 섹션 내 간격 |
| 24dp | 섹션 간 구분 |
| 32dp | 큰 영역 구분 |

---

## 6. 세이프 에어리어 (Safe Area)

| 영역 | 규칙 |
|--------|------|
| 상단 (노치 / 다이내믹 아일랜드) | 모든 HUD 요소는 세이프 에어리어 상단 경계 아래에서 시작 |
| 하단 (홈 바) | 하단 네비게이션 바는 세이프 에어리어 하단 경계에 맞춤 |
| 배경 이미지 | 화면 끝까지 채우는 것(bleed) 허용 (상호작용 없는 경우) |

화면 방향: **세로 모드 고정(Portrait locked).**
